    Installation

    L'application est prévue pour une utilisation locale.

    Installation¶

    1. Cloner le dépôt¶

    git clone https://github.com/Carteetliens/photomapon.git
cd photomapon

    2. Installer les dépendances¶

    pip install -r requirements.txt

    3. Lancer l’application¶

    streamlit run main.py

    Optionnel : vous pouvez également exécuter launch.bat pour automatiser le lancement (à garder dans le dossier de l'interface, créer un raccourci pour l'exécuter ailleurs) 

    @echo off
streamlit run [chemin_vers_main.py]/main.py
pause

    4. Utiliser les données de test¶

    Pour utiliser les données test, référez-vous à la page Principe de base
    Copiez simplement le chemin du dossier /data_tes dans l'application : [chemin_vers_dossier/data_set]
